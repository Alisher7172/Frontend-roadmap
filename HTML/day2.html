Day 2: CSS Styling

Learning the fundamentals of Cascading Style Sheets

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of HTML documents. It controls the visual appearance of web pages, including colors, fonts, layouts, and animations.

CSS allows you to separate content from presentation, making your websites more maintainable and flexible.

CSS Selectors

CSS selectors are patterns used to select and style HTML elements:

  • Element selectors: p, h1, div
  • Class selectors: .className
  • ID selectors: #idName
  • Universal selector: *

Typography

CSS gives you complete control over text styling:

  • Font family, size, and weight
  • Text color and decoration
  • Line height and letter spacing
  • Text alignment and transformation

CSS Best Practices

  • Use meaningful class and ID names
  • Keep CSS organized and commented
  • Use external stylesheets for better maintainability
  • Consider responsive design from the start
  • Optimize for performance
CSS is not just about making things look pretty. It's about creating intuitive, accessible, and engaging user experiences.
