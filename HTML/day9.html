Conditional Statements

If/Else Statements

Execute code based on conditions

let temperature = 25; if (temperature > 30) { console.log("It's hot outside!"); } else if (temperature > 20) { console.log("It's a nice day."); } else { console.log("It's cold outside!"); }

Switch Statements

Execute different code blocks based on a value

let day = "Monday"; switch(day) { case "Monday": console.log("Start of the work week"); break; case "Friday": console.log("Weekend is coming!"); break; default: console.log("It's a regular day"); }

Ternary Operator

Shorthand for simple if/else statements

let age = 20; let canVote = age >= 18 ? "Yes" : "No"; console.log(`Can vote? ${canVote}`);

Loops

For Loop

Execute code a specific number of times

// Count from 1 to 5 for (let i = 1; i <= 5; i++) { console.log(`Count: ${i}`); } // Loop through an array const colors = ['red', 'green', 'blue']; for (let i = 0; i < colors.length; i++) { console.log(colors[i]); }

While Loop

Execute code while a condition is true

let count = 1; while (count <= 5) { console.log(`While count: ${count}`); count++; } // Be careful of infinite loops!

Do-While Loop

Execute code at least once, then repeat while condition is true

let number; do { number = Math.floor(Math.random() * 10); console.log(`Generated: ${number}`); } while (number !== 7); console.log("Finally got a 7!");

Coding Challenge: FizzBuzz

Write a program that prints the numbers from 1 to 100:

  • For multiples of 3, print "Fizz" instead of the number
  • For multiples of 5, print "Buzz" instead of the number
  • For numbers which are multiples of both 3 and 5, print "FizzBuzz"
  • Otherwise, print the number

Solution:

for (let i = 1; i <= 100; i++) { if (i % 15 === 0) { console.log("FizzBuzz"); } else if (i % 3 === 0) { console.log("Fizz"); } else if (i % 5 === 0) { console.log("Buzz"); } else { console.log(i); } }
