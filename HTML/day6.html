Why Learn Modern Layout Techniques?

Responsive Design

Flexbox and Grid make it easy to create layouts that work on all device sizes without complex media queries.

/* Flexbox responsive menu */ .nav { display: flex; flex-wrap: wrap; justify-content: center; }

Complex Layouts

Create advanced magazine-style layouts, masonry grids, and asymmetrical designs with minimal code.

/* Grid magazine layout */ .magazine { display: grid; grid-template-columns: 2fr 1fr; grid-template-rows: 300px auto 200px; gap: 20px; }

Simplified Code

Reduce reliance on floats, clears, and positioning hacks that made CSS layouts difficult in the past.

/* Center with Flexbox */ .center { display: flex; justify-content: center; align-items: center; }

Flexbox Fundamentals

Flex Container

Create a flex container to control the layout of its direct children.

.container { display: flex; /* or inline-flex */ flex-direction: row; /* row, column, row-reverse */ flex-wrap: wrap; /* nowrap, wrap, wrap-reverse */ justify-content: center; /* flex-start, flex-end, space-between */ align-items: center; /* stretch, flex-start, flex-end */ gap: 20px; /* space between items */ }

Flex Items

Properties you can apply to individual flex items.

.item { order: 2; /* change display order */ flex-grow: 1; /* ability to grow */ flex-shrink: 0; /* ability to shrink */ flex-basis: 200px; /* default size */ align-self: flex-end; /* override alignment */ } /* Shorthand */ .item { flex: 1 0 200px; /* grow, shrink, basis */ }

CSS Grid Essentials

Grid Container

Define the grid and its properties.

.container { display: grid; /* or inline-grid */ grid-template-columns: 1fr 2fr 1fr; grid-template-rows: 100px auto 200px; gap: 20px; /* row-gap and column-gap */ grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; }

Grid Items

Place items within the grid using various methods.

.item { /* Line-based placement */ grid-column-start: 1; grid-column-end: 3; grid-row-start: 1; grid-row-end: 2; /* Shorthand */ grid-column: 1 / 3; grid-row: 1 / 2; /* Area-based placement */ grid-area: header; }

Flexbox vs Grid: When to Use Which?

Flexbox

  • Perfect for one-dimensional layouts (row OR column)
  • Ideal for navigation bars, menus, and form elements
  • Great for content distribution and alignment
  • Excellent for responsive elements that wrap
  • Not ideal for complex two-dimensional layouts
  • Can become complex for overlapping elements

CSS Grid

  • Designed for two-dimensional layouts (rows AND columns)
  • Perfect for overall page layouts and complex grids
  • Explicit control over both rows and columns
  • Enables overlapping elements with z-index control
  • Overkill for simple linear components
  • Slightly less browser support for older browsers

Best Practice: Use Both Together!

Flexbox and Grid work perfectly together. Use Grid for the overall page layout, and Flexbox for individual components within the grid.

/* Example: Grid container with flex children */ .page { display: grid; grid-template-columns: 250px 1fr; } .card-container { display: flex; flex-wrap: wrap; gap: 20px; } .card { flex: 1 1 300px; }
