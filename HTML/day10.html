Function Types

Function Declarations

Named functions that are hoisted to the top

// Function declaration function greet(name) { return `Hello, ${name}!`; } // Can be called before declaration console.log(greet("Alice"));

Function Expressions

Functions assigned to variables (not hoisted)

// Function expression const greet = function(name) { return `Hello, ${name}!`; }; // Must be defined before calling console.log(greet("Bob"));

Arrow Functions

Concise syntax (ES6) with lexical 'this' binding

// Arrow function const greet = (name) => `Hello, ${name}!`; // Implicit return const add = (a, b) => a + b; // No 'this' binding const person = { name: "John", greet: () => console.log(this.name) // undefined };

Try It Yourself:
// Function declaration function square(x) { return x * x; } // Function expression const cube = function(x) { return x * x * x; }; // Arrow function const power = (base, exponent) => base ** exponent; // Test the functions console.log("Square of 5:", square(5)); console.log("Cube of 3:", cube(3)); console.log("2 to the power of 8:", power(2, 8)); // Higher-order function example function applyOperation(x, operation) { return operation(x); } console.log("Apply square to 4:", applyOperation(4, square)); console.log("Apply cube to 2:", applyOperation(2, cube));

Scope & Closures

Global Scope

Variables declared outside any function

// Global scope const globalVar = "I'm global"; function checkScope() { console.log(globalVar); // Accessible }

Function Scope

Variables declared inside a function

function example() { // Function scope let functionVar = "I'm in function scope"; console.log(functionVar); } // console.log(functionVar); // Error - not accessible

Block Scope

Variables declared with let/const inside blocks

if (true) { // Block scope let blockVar = "I'm in block scope"; console.log(blockVar); } // console.log(blockVar); // Error - not accessible

Closures

Functions that remember their lexical scope

function createCounter() { let count = 0; return function() { count++; return count; }; } const counter = createCounter(); console.log(counter()); // 1 console.log(counter()); // 2

Closure Code:
// Global scope const globalVar = "Global Value"; function outerFunction() { // Outer function scope const outerVar = "Outer Value"; function innerFunction() { // Inner function scope const innerVar = "Inner Value"; // Closure captures outerVar and globalVar return `${innerVar}, ${outerVar}, ${globalVar}`; } return innerFunction; } const closure = outerFunction(); console.log(closure()); // "Inner Value, Outer Value, Global Value"

Closure Challenge

Create a function that generates private counters:

  • Implement createCounter() that returns an object with increment, decrement, and getValue methods
  • The count should be private and only accessible through the methods
  • Start with an initial value of 0
  • Test by creating two counters that operate independently

Solution:

function createCounter() { let count = 0; return { increment: function() { count++; }, decrement: function() { count--; }, getValue: function() { return count; } }; } const counter1 = createCounter(); counter1.increment(); counter1.increment(); console.log(counter1.getValue()); // 2 const counter2 = createCounter(); counter2.decrement(); console.log(counter2.getValue()); // -1
