JavaScript: Variables, Data Types & Operators

Variables Data Types Operators Quiz

Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const.


        let name = "Alice"
        const PI = 3.14;
        var age = 25;

Data Types

JavaScript supports different data types:

  • String - let city = "London";
  • Number - let score = 99.5;
  • Boolean - let isActive = true;
  • Null - let data = null;
  • Undefined - let x;
  • Object - let person = { name: "Bob", age: 30 };
  • Array - let colors = ["red", "green", "blue"];

Operators

Operators are used to perform operations on variables and values. Some common operators:

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, -=, *=, /=
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !
let a = 10;
let b = 3;
let sum = a + b; // 13
let isEqual = a === b; // false
let isBig = a > b && b < 5; // true
