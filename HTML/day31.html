What is the Fetch API?

Fetch API Basics

The Fetch API is a modern way to make HTTP requests in JavaScript. It returns a Promise and is used to get data from servers.

// Basic fetch fetch('https://jsonplaceholder.typicode.com/users') .then(response => response.json()) .then(data => console.log(data));

What is JSON?

JSON (JavaScript Object Notation) is a lightweight data format for storing and exchanging data. It's easy to read and write for humans and machines.

// Example JSON { "name": "Alice", "age": 25, "email": "alice@example.com" }
