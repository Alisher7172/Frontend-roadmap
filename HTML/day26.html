What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. JavaScript can access and modify the DOM to create interactive web pages.

// Access the DOM
document.body.style.background = "#f0f0f0";

Selecting Elements

  • By ID: document.getElementById('myId')
  • By Class: document.getElementsByClassName('myClass')
  • By Tag: document.getElementsByTagName('div')
  • Query Selector: document.querySelector('.myClass')
const el = document.querySelector('#main');
const items = document.querySelectorAll('.item');

Modifying the DOM

  • Change Text: el.textContent = 'Hello!'
  • Change HTML: el.innerHTML = '<b>Bold!</b>'
  • Change Style: el.style.color = 'red'
  • Add/Remove Class: el.classList.add('active')
  • Create/Remove Element: document.createElement('div')
const newDiv = document.createElement('div');
newDiv.textContent = 'I am new!';
document.body.appendChild(newDiv);

Events

Events let you respond to user actions like clicks, input, and more.

  • Click: el.addEventListener('click', fn)
  • Input: el.addEventListener('input', fn)
button.addEventListener('click', function() {
alert('Button clicked!');
});
