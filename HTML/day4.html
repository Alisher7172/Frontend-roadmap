Day 4 — CSS Basics

Starting CSS

CSS (Cascading Style Sheets) controls how HTML elements look — colors, layout, spacing, fonts and more.

CSS Syntax

Rules consist of a selector and a declaration block:

selector {
    property: value;
    property: value;
}

Example:

p {
    color: #333;
    line-height: 1.5;
}

Selectors

  • Element — targets elements by name: div, p
  • Class — reusable groups: .card
  • ID — single unique element: #main
  • Descendant — target within another: .nav a

Example: <div class="box"></div> and CSS .box { ... }

Box model

Every element is a rectangle: content + padding + border + margin.

Content
Inner padding

margin — space outside border

border — edge line

padding — space around content

Colors & Typography

Colors can be written as hex, rgb, or named values. Control fonts with font-family, size and weight.

Ways to apply CSS

  1. External stylesheet — <link rel="stylesheet" href="styles.css"> (best practice)
  2. Internal <style> block inside <head>
  3. Inline using the style="..." attribute (least preferred)
