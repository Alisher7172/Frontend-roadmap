Array Basics

Creating Arrays

Different ways to create arrays in JavaScript

// Array literal const fruits = ['Apple', 'Banana', 'Cherry']; // Array constructor const numbers = new Array(1, 2, 3); // Array of 5 empty slots const emptyArray = new Array(5); // Array from string const letters = Array.from('Hello');

Accessing Elements

Working with array indexes

const colors = ['red', 'green', 'blue']; // Access by index console.log(colors[0]); // 'red' console.log(colors[2]); // 'blue' // Array length console.log(colors.length); // 3 // Last element console.log(colors[colors.length - 1]); // 'blue'

Modifying Arrays

Adding, removing, and changing elements

let animals = ['dog', 'cat']; // Add to end animals.push('bird'); // ['dog', 'cat', 'bird'] // Remove from end animals.pop(); // ['dog', 'cat'] // Add to start animals.unshift('fish'); // ['fish', 'dog', 'cat'] // Remove from start animals.shift(); // ['dog', 'cat'] // Modify element animals[1] = 'elephant'; // ['dog', 'elephant']

Essential Array Methods

map()

Creates a new array by transforming every element
const nums = [1, 2, 3]; const doubled = nums.map(n => n * 2); // [2, 4, 6]

filter()

Creates a new array with elements that pass a test
const nums = [1, 2, 3, 4]; const evens = nums.filter(n => n % 2 === 0); // [2, 4]

reduce()

Reduces the array to a single value
const nums = [1, 2, 3, 4]; const sum = nums.reduce((t, n) => t + n, 0); // 10

find()

Returns the first element that satisfies a test
const users = [ {id: 1, name: 'John'}, {id: 2, name: 'Jane'} ]; const user = users.find(u => u.id === 2); // {id: 2, name: 'Jane'}

forEach()

Executes a function for each array element
const nums = [1, 2, 3]; nums.forEach(n => console.log(n)); // 1, 2, 3

sort()

Sorts the elements of an array
const nums = [3, 1, 4, 2]; nums.sort((a, b) => a - b); // [1, 2, 3, 4]

slice()

Returns a shallow copy of a portion of an array
const nums = [1, 2, 3, 4, 5]; const part = nums.slice(1, 4); // [2, 3, 4]

splice()

Changes the contents of an array by removing or replacing elements
const nums = [1, 2, 3, 4, 5]; nums.splice(1, 2, 'a', 'b'); // nums becomes [1, 'a', 'b', 4, 5]

Array Challenge

Process this array of products:

const products = [ { id: 1, name: 'Laptop', price: 999, category: 'Electronics' }, { id: 2, name: 'Desk Chair', price: 199, category: 'Furniture' }, { id: 3, name: 'Smartphone', price: 799, category: 'Electronics' }, { id: 4, name: 'Book', price: 15, category: 'Books' }, { id: 5, name: 'Monitor', price: 299, category: 'Electronics' } ];

Requirements:

  • Filter electronics products
  • Apply 10% discount to electronics
  • Sort discounted products by price (high to low)
  • Calculate total value of discounted electronics

Solution:

// Filter electronics const electronics = products.filter( product => product.category === 'Electronics' ); // Apply 10% discount const discounted = electronics.map(product => ({ ...product, price: product.price * 0.9 })); // Sort by price (high to low) discounted.sort((a, b) => b.price - a.price); // Calculate total value const totalValue = discounted.reduce( (sum, product) => sum + product.price, 0 ); console.log("Discounted Electronics:", discounted); console.log("Total Value:", totalValue.toFixed(2));
