Conditionals

Conditionals let you run code based on certain conditions. They are essential for making decisions in your programs. The most common are if, else if, else, and switch:

  • if: Runs a block of code if a condition is true.
  • else if: Checks another condition if the previous if was false.
  • else: Runs if none of the previous conditions were true.
  • switch: Selects one of many code blocks to run, based on a variable's value.

Conditionals help your code "think" and respond to different situations, like checking user input or handling different cases in a game.

let age = 18;
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

let fruit = "apple";
switch (fruit) {
  case "apple":
    console.log("It is an apple");
    break;
  case "banana":
    console.log("It is a banana");
    break;
  default:
    console.log("Unknown fruit");
}

Loops

Loops let you repeat code multiple times, which is useful for tasks like processing lists or running animations. The most common are for, while, and do...while:

  • for: Best when you know how many times you want to repeat something.
  • while: Repeats as long as a condition is true. Good when you don't know the exact number of repetitions in advance.
  • do...while: Like while, but always runs the code at least once before checking the condition.

Loops help you avoid repeating code manually and make your programs more efficient and powerful.

for (let i = 0; i < 5; i++) {
  console.log(i);
}
let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}
let x = 0;
do {
  console.log(x);
  x++;
} while (x < 2);
