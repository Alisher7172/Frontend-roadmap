1. let & const vs var

Block-scoped declarations that solve many issues with var:

  • let - for variables that can be reassigned
  • const - for constants (cannot be reassigned)
  • Both are block-scoped (unlike var which is function-scoped)

2. Arrow Functions

Shorter syntax for functions with automatic this binding:

  • Concise syntax: (a, b) => a + b
  • Inherit this from enclosing scope
  • Perfect for callbacks and array methods
// Will be populated by JavaScript

3. Destructuring Assignment

Extract values from arrays and objects into variables:

  • Array destructuring: [a, b] = [1, 2]
  • Object destructuring: {name, age} = user
  • Default values and renaming supported

4. Spread & Rest Operators

Three dots with different meanings based on context:

  • Spread: Expand arrays/objects [...arr]
  • Rest: Collect parameters (...args)
  • Copy arrays, merge objects, function parameters

5. Template Literals

Enhanced strings with backticks for interpolation and multiline:

  • Variable interpolation: `Hello ${name}!`
  • Multiline strings without escape characters
  • Embedded expressions and function calls
